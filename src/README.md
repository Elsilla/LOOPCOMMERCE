# LoopCommerce - E-Commerce Simulator (Consola en Java)

## 2. Descripción breve del desafío
En esta actividad, se desarrollará una aplicación de consola llamada **LoopCommerce**, que simula las operaciones básicas de un e-commerce.

El objetivo es demostrar dominio de:

- `while`
- `do-while`
- `for`
- `foreach`
- uso de **Scanner**
- manejo de variables, operadores y control de flujo

## ✔ 3. Tecnologías usadas

- Java 21
- Scanner
- Loops (for, foreach, while, do-while)

## ✔ 4. Cómo ejecutar el proyecto

Ejemplo:

```
javac LoopCommerce.java
java LoopCommerce
```

## ✔ 5. Funcionalidades del programa

- Primero se despliega la informacion de los productos y sus precios, estan en arreglos distintos lo que puede provocar
que si se ejecuta algun cambio en estos, no coincidan necesariamente productos y valor.


- Despues mediante un scanner es posible seleccionar los productos y acumularlos pudiendo ver el resultado de la suma de
sus valores.


- mediante la opcion 3 es posible comprobar dicho subtotal.



- Por ultimo se crea un arreglo con strings que simulan un proceso de compra mediante un `bucle for` y un `Thread.sleep` 
para añadir mejor estetica poe terminal.
